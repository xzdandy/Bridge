Introduction

Let's start with the basic opening structure of the system:

  • ∅
    • 1♣
      Polish club, one of three
    • a) 11–14 HCP, BAL, 2+♣; We open 1♣ for any balanced hands including hands with 4♦. With 5♦ and a balanced hand (5332), we open 1♦. We also need to open 1♣ for 4♠-4♥-1♦-4♣ hand, and for other 4441 hands we open 1♦. So the preparatory version of 1♣ opening promises 2+♣, and denies any 5-card suits except for the club.
    • b) 15–17 HCP, UNBAL, 5+♣; The natural version of 1♣ opening always promises a unbalanced hand. For 5422 hands, we try to open 1NT. So if we choose to open 1♣, it usually means points are concentrating in long suits and lacking stoppers in the 2-card suits.
    • c) 18+ HCP, any distribution; Upgrading a good distribution hand with fewer points is allowed. In other words, if it is likely to make game with partner owning fewer than 7 points, we consider opening the strong 1♣.
    • 1♦
      11–17 HCP, 5+♦; The 1♦ opening usually promises 5+♦, and only with 2 exceptions.
    • a) All 4441 hands with 4♦.
    • b) 11–14 HCP, 4+♦-5+♣.
    • 1♥♠
      11–17 HCP, 5+♥ / 5+♠
    • 1NT
      15–17 HCP, BAL; We try to open 1NT as much as possible, so it can be semi-balanced.
    • 2♣
      11–14 HCP, 6+♣ / 5+♣-4M; For the hand with exactly 5♣ and 4M, we open 1♣ if considering it balanced from the view of points, so we still can end in declaring 1NT.
    • 2♦
      4–9 HCP, 6♥ / 6♠
    • 2♥♠
      4–9 HCP, 5♥-4+m / 5♠-4+m; When vulnerable, it has to be 5-5 and the 2♥ opening can be 5♠-5♥ (For the 2♠ opening, the second suit is always minor).
    • 2NT
      4–9 HCP, 5♦-5♣
    • 3X
      PRE; Destructive.
    • 3NT
      Gambling; Solid 7-card minor suit and nothing else.
    • 4♣♦
      PRE; Even more destructive.
    • 4♥♠
      7+M, T/P; We can only raise to 6M, if owning 3+ Aces and a good hand.
    • 4NT
      BW
    • 5♣♦
      8+m, T/P; The same as the 4M opening.
    • 5♥♠
      7+M, INV, S/T; We raise to 6M with one trump honor, and raise to 7M with two trump honors.
    • 5NT
      6+♦-6+♣, F; Pick a slam in minor. 7m is applicable if we have one trump honor and good support for the other minor (shortness with trump length to ruff or queen).
    • 6X
      T/P; We raise to 7X with one trump honor (A or K).
    • 6NT
      6+♦-6+♣, F; Pick a grand slam in minor.
    • 7X
      T/P

The 1♣ opening

The feature of this variation is 1NT invitation after 1♣ opening. The idea is we try to either play 1NT or 3NT.

The 1♦ opening

  • 1♦
    • 1♥♠
      6+ HCP, 4+♥ / 4+♠, F1; With 4-4 in the major, we response 1♥ first. With 5-5 in the major, we response 1♠ first, but only with 10+ HCP. With unequal length in the major, we response the longer one first. Notice the direct jump 2♥ response after the 1♦ opening.
    • 1NT
      8–10 HCP, NF; This denies 4-card major suits, but this might not be balanced.
    • 2♣
      5+♣, FG; Different from 1M opening, 2♣ after 1♦ opening always promises 5+♣, as we have plenty of other bids to show other types of hands.
    • 2♦
      10+ HCP, 4+♦, F; This denies 4-card major suits and is forcing to 3♦.
    • 2♥
      6–9 HCP, 5♠, 4+♥;
    • 2♠
      11–12 HCP, INV to 3NT; This denies 4-card major suits and 4♦, so it usually implies a few clubs.
    • 2NT
      13–15 HCP, FG to 3NT; The same as the 2♠ response, but this one is game forcing. With the help of the 2♠ response, we are able to create a balanced game forcing at 2 level. So in the case that the opener has an interesting hand, he still can bid the distribution at 3 level.
    • 3♣
      9–11 HCP, 6♣, NF; This denies the 3-card diamond fit and 4-card major suits.
    • 3♦

    • 3♥♠

The following abbreviations or terms may be used (Note the use of BLOCK CAPITALS and SLASHES (/)):

  • (5431) = Any hand with that distribution (suits unknown)
  • 5431 = Five spades, four hearts, three diamonds, one club
  • 54(31) = A hand with five spades, four hearts, and 3♦1♣ or 3♣1♦
  • 54(xx) = A hand with five spades and four hearts
  • AGG = Aggressor. The first player to double or overcall for the defending side
  • ADV = Advancer, aggressors partner
  • ASK = Asking bid
  • ART = Artificial
  • ATT = Attitude
  • B = Black suit(s)
  • BAL = Balanced
  • BW = Blackwood
  • CB = Checkback
  • COMP = Competitive
  • CONC = Concentrated (e.g. all values in the bid suits)
  • CONST = Constructive
  • CTRL = Control
  • CUE = Cue-bid
  • DISC = Discourage (ing)
  • E = Even
  • ENC = Encourage (ing)
  • FRAG = Fragment
  • F = Forcing
  • F1 = Forcing 1 round
  • F2NT = Forcing to 2NT
  • FG = Forcing to game
  • 4♠F = 4th suit forcing (4♠FG, 4♠F1)
  • FREQ = Frequent
  • G/T = Game try
  • H = Honour (Ace, King, or Queen)
  • HCP = High Card Points
  • INV = Invitational
  • INQ = Inquiry
  • KCB = Keycard Blackwood
  • L/D = Lead-directing
  • LEB = lebensohl
  • LHO = The opponent on your left
  • LIM = Limit raise
  • L/S = Long suit
  • L/T = Less than (length or strength)
  • M = Major
  • MM = majors
  • m = Minor
  • mm = minors
  • MAX = Maximum, Maximal, Maximal Overcall Double
  • MIN = Minimum
